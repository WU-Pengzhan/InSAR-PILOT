# Security Policy

## Supported versions

Security fixes are provided for the latest released version of InSAR-PILOT.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report vulnerabilities privately through GitHub's
[private vulnerability reporting](https://docs.github.com/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability):
go to the repository's **Security** tab and select **Report a vulnerability**.

We will acknowledge your report, investigate, and coordinate a fix and
disclosure timeline with you.
